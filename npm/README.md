# airoute-cli

This package installs the platform-specific binary for [AI Router](https://github.com/soooooollee/ai-router).

```bash
npm install --global airoute-cli
air init
air start
```

Supported platforms are macOS, Linux, and Windows on x64 or ARM64.

Set `AIROUTE_VERSION` to install a specific release or `AIROUTE_NPM_SKIP_DOWNLOAD=1` when packaging without downloading the binary.
